Nearest Taco Time

A D3 map of all Taco Time Northwest franchises.

To make this map, I borrowed liberally from Mike Bostock's Let's make a map, and from his block for U.S. Airports Voronoi. I wrangled address data from the Find a Taco Time and munged the data into lat-lon coordinates using Open Street Map's Nominatim tool.

  • Date: March 2015
  • Skills used: D3, APIs, Python, OGR

Ishmael Calling

A Markov bot which reads Moby Dick so you don't have to.

Ishmael Calling is a simple twitter Markov bot that reads in the full text of Moby Dickfrom Project Gutenberg, makes random markov chains sentences from that text, then posts 140-characters results to twitter.

  • Date: December 2015
  • Skills used: Python, twitter API
